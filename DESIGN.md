# Design System Documentation


## Theme Overview


This design system uses a **dark mode** color scheme to provide a sophisticated and comfortable visual experience, especially in low-light environments. The overall aesthetic is balanced with **moderate roundedness** for UI elements, giving them a contemporary and approachable feel. Spacing is set to a **normal** level, ensuring good readability and a clean layout without feeling too dense or overly sparse.


## Typography


The typography is consistently set to **Inter** for all text elements.

- **Headlines:** Inter

- **Body Text:** Inter

- **Labels:** Inter


This choice promotes visual unity and ensures a clear, modern, and highly legible text presentation across the entire system.


## Color Palette


The core color palette is defined by specific hex values for primary, secondary, tertiary, and neutral colors, providing a strong visual identity.



- **Primary Color:** `#1e293b` - This deep, dark blue serves as the foundational brand color, suitable for primary actions and key interactive elements.

- **Secondary Color:** `#0f3b5e` - A slightly lighter, yet still dark, blue that complements the primary color, used for supporting UI elements and secondary actions.

- **Tertiary Color:** `#10b981` - A vibrant green acting as an accent color, providing visual highlights, badges, and decorative elements.

- **Neutral Color:** `#0f172a` - A very dark, almost black, neutral base color used for backgrounds, surfaces, and non-chromatic elements, reinforcing the dark mode theme.


## Corner Roundness


UI elements feature **moderate roundedness (2)**. This level of curvature strikes a balance between sharp, angular aesthetics and highly rounded, pill-shaped designs, contributing to a modern and friendly appearance.


## Spacing


The design system employs **normal spacing (2)**. This ensures a comfortable amount of whitespace between and within UI elements, enhancing readability and overall visual clarity without creating an overly spacious or compact interface.
